# 프로젝트 요구사항 (요약)

## 목표
- 사용자가 최소한의 입력으로 업무를 기록/관리/추적하도록 자동화
- 웹/모바일 반응형 기반의 심플하고 트렌디한 UI
- 이모지/아이콘을 적극 활용한 직관적 시각화
- 무료 플랫폼만 활용 가능한 기술 스택 전제

## 핵심 기능 (필수)
1. 스마트 우선순위 자동 분류
2. 데일리 요약 알림(자동 리포트)
3. 진행률 시각화 대시보드
4. 반복 업무 템플릿(체크리스트 자동 생성)
5. 문구 점검 & 개선 제안
6. 연락처 기반 작업 연결
7. 지연 감지 알림(병목 탐지)
8. "다음 액션" 자동 제안
9. 업무 담당자 지정(담당자/공유/협업 연결)
10. 업무 리마인드 주기 자동 조정

## 주요 화면
- 홈 대시보드
- 업무 상세
- 알림 센터
- 연락처/담당자 보기
- 설정

## 운영 원칙
- 무료 플랜 기반 구현 우선
- 구현 불가 시 대체안 2개 제시
- 접근성(대체 텍스트/ARIA) 포함
