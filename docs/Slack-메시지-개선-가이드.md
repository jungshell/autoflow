# Slack 메시지 개선 가이드

## 구현된 내용

### 현재 Slack 메시지 구조

```
📋 *AutoFlow 데일리 요약*

📊 *통계*
• 총 업무: 12건
• 완료율: 17%
• 지연: 7건

🚨 *긴급한 일 (3건)*
1. 🔴 긴급 업무 1 (1/27 15:00)
2. 🟠 높은 우선순위 업무 (1/27 18:00)
...

📅 *오늘 해야 할 일 (5건)*
1. 🟡 중간 우선순위 업무 (1/27 16:00)
...

⏰ *3일 내 끝내야 할 일 (4건)*
1. 🟢 낮은 우선순위 업무 (1/30 10:00)
...

⚠️ *지연된 업무 (2건)*
1. 지연된 업무 1 (1/25 14:00)
...
```

---

## 추가로 유용한 정보 제안

### 1. 완료된 업무 (오늘)
- 오늘 완료한 업무 목록
- 동기 부여 및 성취감 제공

### 2. 진행 중인 업무
- 현재 진행 중인 업무 상태
- 진행률 표시 (가능한 경우)

### 3. 담당자별 업무 분포
- 누가 어떤 업무를 맡고 있는지
- 업무 분산 확인

### 4. 마감일이 없는 업무
- 마감일이 설정되지 않은 업무 알림
- 빠뜨리기 쉬운 업무 방지

### 5. 주간 트렌드
- 지난 주 대비 완료율 변화
- 생산성 추이 파악

### 6. 다음 액션 제안
- 가장 우선순위 높은 3가지 업무
- 즉시 시작할 수 있는 업무

### 7. 업무 분류별 통계
- 우선순위별 분포
- 상태별 분포 (todo/in_progress/done)

### 8. 연락처별 업무
- 특정 연락처와 관련된 업무
- 협업 업무 추적

---

## 우선순위별 추천

### 높은 우선순위 (즉시 추가 권장)
1. ✅ **완료된 업무 (오늘)** - 동기 부여
2. ✅ **마감일이 없는 업무** - 놓치기 쉬운 업무 방지
3. ✅ **다음 액션 제안** - 즉시 시작할 업무

### 중간 우선순위
4. 진행 중인 업무 상태
5. 담당자별 업무 분포

### 낮은 우선순위 (나중에 추가 가능)
6. 주간 트렌드
7. 업무 분류별 통계

---

## 현재 구현된 기능

- ✅ 오늘 해야 할 일 목록
- ✅ 3일 내 끝내야 할 일 목록
- ✅ 긴급한 일 목록 (지연 + urgent + 24시간 이내)
- ✅ 지연된 업무 목록
- ✅ 통계 정보 (완료율, 총 업무, 지연 건수)
- ✅ 우선순위 이모지 표시 (🔴🟠🟡🟢)
- ✅ 마감일 시간 표시

---

## 메시지 포맷 특징

- **구조화된 섹션**: 각 카테고리별로 명확히 구분
- **이모지 사용**: 시각적 구분 및 가독성 향상
- **우선순위 표시**: 색상 이모지로 우선순위 표시
- **마감일 표시**: 각 업무의 마감일 시간 표시
- **최대 5개 표시**: 너무 길지 않게 제한

---

## 향후 개선 방향

### Slack Block Kit 사용 (선택사항)
현재는 텍스트 포맷이지만, Slack Block Kit을 사용하면:
- 버튼 추가 (업무 완료, 상세 보기 등)
- 이미지 첨부
- 인터랙티브 요소 추가

### 개인화
- 사용자별 맞춤 메시지
- 관심 있는 업무만 표시

### 알림 설정
- 특정 조건에서만 알림 (예: 긴급 업무가 있을 때만)
- 알림 빈도 조절
