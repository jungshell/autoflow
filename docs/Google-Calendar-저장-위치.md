# Google Calendar 저장 위치

## 생성되는 일정이 저장되는 캘린더

AutoFlow에서 생성되는 업무 일정은 **Google Calendar의 기본 캘린더(primary calendar)**에 저장됩니다.

### 코드 위치
- 파일: `src/lib/googleCalendar.ts`
- API 엔드포인트: `https://www.googleapis.com/calendar/v3/calendars/primary/events`

### 기본 캘린더란?

- Google 계정의 **주 캘린더**입니다
- Google Calendar 앱/웹에서 가장 기본적으로 표시되는 캘린더입니다
- 일반적으로 사용자의 이름과 동일한 이름을 가집니다 (예: "홍길동의 캘린더")

### 확인 방법

1. **Google Calendar 웹/앱 접속**
2. 왼쪽 사이드바에서 **"내 캘린더"** 섹션 확인
3. 가장 위에 있는 캘린더가 기본 캘린더입니다
4. AutoFlow에서 생성한 일정은 여기에 `[업무] 제목` 형식으로 표시됩니다

### 일정 형식

- **제목**: `[업무] {업무 제목}`
- **설명**: 업무 설명 + 업무 ID
- **기간**: 마감일 기준 1시간
- **알림**: 
  - 1일 전 이메일 알림
  - 1시간 전 팝업 알림

### 다른 캘린더에 저장하고 싶다면?

현재는 기본 캘린더에만 저장되도록 구현되어 있습니다. 특정 캘린더에 저장하려면:
1. Google Calendar API에서 캘린더 목록 조회
2. 사용자가 캘린더 선택 UI 제공
3. 선택한 캘린더 ID로 이벤트 생성

이는 향후 기능 개선 사항으로 추가할 수 있습니다.
