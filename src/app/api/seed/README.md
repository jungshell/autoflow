# 테스트 데이터 관리

## 테스트 데이터 추가

```bash
curl -X POST http://localhost:4000/api/seed
```

또는 브라우저에서 직접 접속:
- GET 요청은 지원하지 않습니다. POST 요청만 가능합니다.

## 테스트 데이터 삭제

```bash
curl -X DELETE http://localhost:4000/api/seed/clear
```

## 생성되는 테스트 데이터

### 연락처 (5명)
- 민지 (디자인팀)
- 유진 (마케팅팀)
- 서현 (개발팀)
- 태훈 (기획팀)
- 나 (본인)

### 업무 (10건)
- 긴급: 마감 임박 제안서 제출 (6시간 후)
- 높음: 고객 미팅 정리 및 후속 연락 (12시간 후)
- 중간: 브랜드 문구 점검 및 수정 (내일)
- 높음: 월간 보고서 작성 (2일 지연)
- 중간: 웹사이트 리뉴얼 디자인 검토 (3일 후)
- 낮음: 마케팅 캠페인 기획안 작성 (일주일 후)
- 중간: 고객 피드백 수집 및 분석 (5일 후)
- 완료: 완료된 프로젝트 문서화 (어제)
- 완료: 팀 미팅 준비 자료 (2시간 전)
- 높음: 신규 고객 온보딩 프로세스 개선 (2일 후)

### 알림 (4건)
- 지연 감지 알림
- 리마인드 주기 조정 알림
- 데일리 요약 알림
- 제안 알림

### 템플릿 (3건)
- 주간 업무 점검 체크리스트
- 월간 보고서 요약 템플릿
- 고객 미팅 준비 체크리스트
